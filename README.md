🕒 Gestor de Horas de Estágio
Uma ferramenta simples, intuitiva e interativa desenvolvida em HTML, CSS e JavaScript puro para ajudar estudantes a gerenciarem suas horas de estágio. Chega de fazer contas complexas na mão para descobrir quando você finalmente vai bater a sua meta de horas!

✨ Principais Funcionalidades
Previsão Exata: Insira a carga horária total exigida (ex: 260h, 300h) e a data de início para descobrir a data exata de término do seu estágio.

Configuração de Turnos: Defina seus horários de entrada e saída (com suporte a até 2 turnos por dia) para cada dia da semana.

Calendário Interativo: O sistema gera um calendário completo. Teve um imprevisto ou é feriado? Basta clicar no dia verde para marcá-lo como falta (vermelho), e a calculadora empurrará sua data final para frente automaticamente.

Alertas Inteligentes (Lei do Estágio): A ferramenta emite alertas informativos caso a sua rotina planejada fira regras básicas da Lei do Estágio (jornadas maiores que 6h diárias, 30h semanais ou ausência do intervalo de 15 minutos para turnos acima de 4h).

Resumo Detalhado: Saiba exatamente sua carga horária semanal, quantos dias efetivos você trabalhará e com quantas horas e minutos você encerrará o seu último dia de expediente.

🚀 Como Usar

Acesse o site: https://makocago.github.io/Calculadora-de-horas-de-estagio/

Ou faça o download ou clone este repositório.

Dê um duplo clique no arquivo index.html para abri-lo em qualquer navegador de internet moderno (Chrome, Edge, Firefox, Safari).

Preencha seus dados e comece a planejar! Não é necessária nenhuma instalação adicional.
